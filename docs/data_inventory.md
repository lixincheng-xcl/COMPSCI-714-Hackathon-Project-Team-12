# Data Inventory

## 1. Purpose

This document tracks the data and reference materials required for the Research Contract Adviser Agent.

## 2. Data Categories

| Data Type | Description | Status | Owner |
|---|---|---|---|
| Sample Contracts | Anonymised or synthetic research contracts for testing | Pending | Ocean / Xincheng |
| Reference Templates | Standard university contract templates | Pending | Ocean |
| Contracting Positions | Standard institutional positions for key clauses | Pending | Ocean |
| Benchmark Cases | Small manually reviewed examples for evaluation | Pending | Ocean |
| Generated Reports | Output reports produced by the system | Pending | Xincheng |

## 3. Clause Categories

| Clause Category | Description |
|---|---|
| Intellectual Property | Ownership, licensing, commercialisation rights |
| Confidentiality | Confidential information and disclosure restrictions |
| Publication | Academic publication rights and review periods |
| Liability | Responsibility, indemnity, damages |
| Data Use | Data handling, sharing, privacy |
| Termination | Contract ending conditions |
| Payment / Funding | Financial obligations and milestones |
| Governing Law | Jurisdiction and applicable law |
| Dispute Resolution | Escalation and dispute process |

## 4. Data Handling Rules

- Do not upload real confidential contracts to GitHub.
- Use anonymised, synthetic, or public sample documents only.
- Store sensitive files outside the public repository.
- Do not commit API keys, private credentials, or identifiable contract data.
