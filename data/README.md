# Data Folder

This folder is used for sample data and reference materials.

## Folder Structure

data/
├── sample_contracts/
├── reference_templates/
├── private/
└── processed/

## Important Notice

Do not upload real, confidential, or non-anonymised contract documents to this repository.

Only use:

- anonymised sample contracts;
- synthetic contracts;
- public reference materials;
- non-sensitive testing files.

## Folder Descriptions

| Folder | Purpose |
|---|---|
| sample_contracts/ | Safe sample contracts for testing |
| reference_templates/ | Standard template clauses or reference materials |
| private/ | Local-only private files. This folder should not be committed |
| processed/ | Processed intermediate data. This folder should not be committed |
